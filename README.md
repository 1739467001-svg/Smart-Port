# 🏗️ 智港数字员工 · SmartPort Digital Workforce

> 以多智能体「数字员工」重构港口作业：当 3D 数字孪生遇上一支永不停歇的 AI 数字劳动力

**智港数字员工** 把一座集装箱码头搬进浏览器——真实的 3D 数字孪生（海面、岸桥、堆场、AGV、船舶），
并在其上运行一支由 **5 名 AI 数字员工** 组成的协同班组：从单证、配载、安全、调度到执行，
覆盖集装箱进出口全流程。每一名数字员工都对应港口里一个真实岗位，可被观察、可被下钻、可被替代。

## 👷 数字员工班组

| 数字员工 | 岗位 | 职责 | 替代的人力 |
|-------|------|------|-----------|
| 📋 单证数字员工 | Documentation Agent | 订舱/报关数据清洗、箱单 EDI 解析与标准化 | 单证员 |
| 📦 配载数字员工 | Stowage Agent | 空间推演与配载优化（真实模拟退火算法） | 配载工程师 |
| 🛡️ 安全数字员工 | Safety Agent | 三维重心 CoG 校验、危品/超限拦截与合规审查 | 安全稽核员 |
| 🚦 调度数字员工 | Dispatch Agent | 岸桥/轨道吊/AGV 协同、作业序列编排与冲突消解 | 调度指挥 |
| ⚡ 执行数字员工 | Execution Agent | 生成机器可读指令集、下发自动化设备并闭环回传 | 现场操作 |

> 五名数字员工以「协同 → 审查 → 共识 → 执行」的方式互相通信（右侧协同日志实时可见），
> 形成一个可解释、可审计的港口 AI 数字劳动力。

## 💡 产业价值（为什么是数字员工）

- **降本增效**：配载数字员工把容积利用率与重心安全一次算到位，替代人工反复试排；
  配载 AI 相较传统人工利用率提升可在 L3 单箱视图中「传统人工 ↔ AI 优化」实时对照。
- **7×24 不间断**：数字员工无需换班，全流程在途箱量与效能指标持续刷新。
- **安全合规**：安全数字员工对每一个配载方案做三维重心校验，不安全方案直接否决。
- **可复制**：一套数字员工班组可平移到任意集装箱码头，岗位即产品、可签约可驻场。

## 🚀 快速开始

### 本地开发

```bash
cd frontend
npm install
npm run dev
```

访问 `http://localhost:5173`

### Docker 部署

```bash
docker compose up --build
```

访问 `http://localhost:3000`

### 云服务器部署

```bash
# 1. 在服务器上 clone 代码
git clone <your-repo-url>
cd smart-port

# 2. 构建并启动
docker compose up -d --build

# 3. 配置 Nginx 反向代理（可选，用于域名+HTTPS）
# 参见 DEPLOYMENT.md
```

## 🎮 操作指南

- **▶ 一键路演** — 顶栏一键启动自动路演：L1→L2→L3 自动巡航，自动演示「传统人工 → AI 配载」翻盘时刻，答辩不怕手抖
- **📊 降本增效** — 顶栏打开数字员工 ROI 测算（年化效益 / 回收期 / 单箱降本，每项均含推导公式）
- **💬 问数字员工** — 顶栏打开自然语言问答：直接问「重心安全吗 / AI 比人工好多少 / 一年省多少钱」，数字员工用实时数据作答（离线，路演零翻车）
- **左键拖拽** — 旋转视角
- **滚轮** — 缩放
- **点击堆场** — 下钻到堆区视图（L2）
- **点击集装箱** — 下钻到单箱装载视图（L3），切换「传统人工 ↔ AI 优化」看配载数字员工的效果
- **点击数字员工光标** — 查看该数字员工的岗位职责、实时指标与协同日志
- **右侧面板** — 查看数字员工班组实时状态
- **PAUSE/RESUME** — 暂停/恢复仿真
